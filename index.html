Proyecto 1

Proyecto creado con Laravel para la gestión de un negocio de sneakers

Estructura

Consiste en una landing page para que el usuario conozca el servicio y productos que ofrece la tienda y una parte de administrador, en la cuál puede prealizar operaciones de gestión de ventas.

Restricciones

Un usuario no puede modificar ni eliminar una venta que no haya sido creada por él.

En caso de eliminar alguna venta, ésta se almacenará en una papelera de reciclaje antes de ser eliminada permanentemente para que en caso de que haya una confusión, pueda ser restaurada correctamente

Cada vez que se realice una venta, se le podrá notificar a quien realizó la venta si fue registrada correctamente y se le enviará una notificación vía email.
